## Requirements

* `Node.js`
* `npm or yarn `

## Setup

1. clone this repos 
  * `$ git clone https://github.com/hoflish/HF_mockup_challenge.git`
  * `cd HF_mockup_challenge `
  
2.  Run `yarn install or npm install`  

3. Browse to [http://localhost:3000](http://localhost:3000)
